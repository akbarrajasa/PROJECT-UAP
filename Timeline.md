# TIMELINE PENGERJAAN (TARGET)

## 1. Hari 1 - 3 : Setup Pemain
* Buat karakter tertentu yang bisa jalan jalan dengan menggunakan keyboard arah di layar kosong
* Buat karakter agar tidak bisa menembus ujung kanan dan kiri

## 2. Hari 4 - 7 : Setup Musuh
* Buat 1 baris musuh
* Buat musuh jalan sendiri dari kiri ke kanan
* Buat Loooping (kalau udah di ujung kanan, dimulai lagi dari ujung kiri)
* Buat duplikasi jadi beberapa baris dengan arah yang berbeda 

## 3. Hari 8 - 10 : Kondisi Game
* Buat kondisi kalah kalau koordinat musuh = koordinat karakter pemain
* Buat kondisi menang kalau koordinat player = koordinat layar paling atas

## 4. Hari 11 - 14 : Finishing
* Buat skor
* Buat printan Game Over dan You Win
